# プライバシーポリシー（MoneyForward Toolpack）

制定日: 2025-12-17

本ポリシーは、Chrome 拡張機能「MoneyForward Toolpack」（以下「本拡張」）における、ユーザーデータの取り扱いについて説明します。

## 1. 収集・利用する情報

本拡張は、MoneyForward ME の家計簿ページ上で次の情報を取り扱う場合があります。

- 取引情報（例: 日付、内容、金額、メモ等）
- 本拡張が付与・管理するラベル情報（例: 利用中／ほぼ未使用／解約予定）
- カテゴリルール（ホワイトリスト／ブラックリスト）および設定値（しきい値、トグル設定等）
- CSV ダウンロード機能で生成されるファイル（ユーザー操作により端末へ保存）

これらは、本拡張の機能（サブスク候補の検出・可視化、ラベル管理、重複チェック、カテゴリルール警告、CSV ダウンロード）を提供する目的でのみ利用します。

## 2. 保存（ローカルストレージ）

本拡張は、設定値やラベル情報、カテゴリルール等を、ブラウザの拡張機能ストレージ（`chrome.storage`）に保存します。

- 本拡張は、開発者が運用する外部サーバーへ当該データを送信・保存しません。
- 端末内に保存されたデータは、拡張機能の削除、またはブラウザ側の拡張機能データ削除により消去できます。

## 3. 外部送信（第三者提供）

### 3.1 Gemini API への送信（任意機能）

本拡張は、ユーザーが「Gemini 解析」を有効化し、かつ解析を実行した場合に限り、サブスクリプション候補判定のために取引情報の一部を Google の Gemini API（Generative Language API）へ送信します。

- 送信は、本拡張の解析機能を提供する目的に限定されます。
- Gemini API の利用には、ユーザー自身が取得・設定する API キーが必要です（本拡張が API キーを発行・収集することはありません）。
- 送信されたデータの取り扱いは、Google 側の利用規約・プライバシーポリシー等に従います。

### 3.2 MoneyForward へのアクセス

本拡張は、MoneyForward ME の家計簿ページ上で動作し、ページ表示に必要な範囲で DOM を参照します。また、CSV ダウンロード機能では、ユーザーの明示操作により、同一タブ内でログインセッション（Cookie）を利用して CSV 取得リクエストを実行します。

## 4. 広告・解析・トラッキング

本拡張は、広告目的のデータ収集や、ユーザー行動のトラッキングを行いません。

## 5. セキュリティ

本拡張は、必要最小限の権限で動作するよう設計し、外部送信が必要な場合は目的を限定します。API キーや設定値はブラウザ内に保存され、開発者が管理するサーバーへ送信されません。
