# MoneyForward Toolpack

MoneyForward ME の家計簿ページで、サブスクリプション（固定費）の可能性が高い取引を Gemini API を用いて検出し、可視化・管理するための Chrome 拡張機能です。

## 🚀 機能

- **AIによる自動解析**: Gemini API (Flash/Pro) を利用して、取引明細からサブスクリプションの可能性をスコアリング。
- **ハイライト表示**: 設定したしきい値を超える「サブスク候補」の行を自動でハイライト。
- **ラベル管理**: 各取引に対して「利用中」「ほぼ未使用」「解約予定」などのステータスをラベル付けして管理。
- **Gemini解析のON/OFFトグル**: オプション画面で解析機能を有効/無効に切り替え可能。OFF時はポップアップの再解析ボタンも自動で無効化されます。
- **プライバシー配慮**: 解析は月単位で手動/自動実行され、結果はブラウザ内に保存されます。

## 📦 インストール方法

1. このリポジトリをクローンまたはダウンロードします。
   ```bash
   git clone https://github.com/yousuke106/moneyForward_Toolpack.git
   ```
2. Google Chrome を開き、拡張機能管理ページ (`chrome://extensions`) にアクセスします。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、クローンしたフォルダを選択します。

## ⚙️ 設定方法

1. 拡張機能のアイコン、またはポップアップ内の「設定を開く」ボタンからオプション画面を開きます。
2. **Gemini APIキー**: [Google AI Studio](https://aistudio.google.com/) でAPIキーを取得し入力します。
3. **モデル**: `gemini-2.5-flash` (推奨) などを選択します。
4. **しきい値**: AIのスコア（0-100）に対する判定ラインを設定します（デフォルト: 70）。
5. **Gemini解析トグル**: 解析を無効化すると自動・手動解析を停止し、ポップアップの再解析ボタンも無効化されます（ラベル機能は利用可）。
