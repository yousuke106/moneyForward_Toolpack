# リリースノート

## 0.1.3 (2025-11-29)

- コンテキストメニューの再登録を直列化し、`mf-download-visible-month` 重複登録による `runtime.lastError` を解消。
- 設定変更時のメニュー更新も安全化（非同期失敗を抑制）。
- ビルド成果物を更新（`dist/` & `dist.zip`）。
- 権限・ホスト権限の追加変更なし。

- サブスクラベルUIをメモ欄から独立した専用列に分離し、満足度機能無効化時も消えないよう列削除ロジックを修正。
