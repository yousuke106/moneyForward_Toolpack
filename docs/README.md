# ドキュメント索引（カテゴリ別）

必要な資料をカテゴリごとに配置しました。各カテゴリのREADMEから詳細へ進んでください。

- `overview/` : プロジェクト全体像・構成
- `requirements/` : 仕様・要件定義
- `plans/` : 実装/リリース計画（`plans/README.md` を参照）
- `checklists/` : 実装・テスト・リリース用チェックリスト

## まず読むもの
- `requirements/mf_subs_requirements_v1.md` — サブスク検出・ラベル管理の正式要件。
- `plans/mf_subs_design_outline.md` — 要件に基づく設計準備メモ（初版）。

## 更新ルール
- 新規ドキュメントを追加したら該当カテゴリのREADMEにリンクを追記する。
- 非推奨になった資料はタイトル末尾に `(deprecated)` を付記し、削除予定日を明記する。
